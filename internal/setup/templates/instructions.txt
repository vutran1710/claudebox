# ClaudeBox Remote Server

You are connected to a ClaudeBox remote server.

## Session API

Base URL: {{.ServeURL}}
Auth: `X-API-Key: {{.ServeKey}}`

| Method | Endpoint | Body | Description |
|--------|----------|------|-------------|
| POST | /sessions | `{name, repo?}` | Create session (finds/clones/creates workspace) |
| GET | /sessions | — | List active sessions |
| DELETE | /sessions/{name} | — | Kill a session |
| GET | /health | — | Health check |

## Message API

Base URL: {{.AMURL}}
Auth: `X-API-Key: {{.AMKey}}`

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/messages | List messages (`?source=gmail`, `?q=search`) |
| GET | /api/messages/{id} | Get single message |
| GET | /api/stats | Counts by source |
| POST | /webhook/chrome-lite-mcp | Webhook for background jobs |

## Examples

Create a session:
```bash
curl -X POST {{.ServeURL}}/sessions \
  -H "X-API-Key: {{.ServeKey}}" \
  -H "Content-Type: application/json" \
  -d '{"name": "my-app"}'
```

Create a session with a repo:
```bash
curl -X POST {{.ServeURL}}/sessions \
  -H "X-API-Key: {{.ServeKey}}" \
  -H "Content-Type: application/json" \
  -d '{"name": "my-app", "repo": "owner/repo"}'
```

List sessions:
```bash
curl {{.ServeURL}}/sessions -H "X-API-Key: {{.ServeKey}}"
```

Kill a session:
```bash
curl -X DELETE {{.ServeURL}}/sessions/my-app -H "X-API-Key: {{.ServeKey}}"
```

Read messages:
```bash
curl "{{.AMURL}}/api/messages?source=gmail" -H "X-API-Key: {{.AMKey}}"
```

Message stats:
```bash
curl "{{.AMURL}}/api/stats" -H "X-API-Key: {{.AMKey}}"
```
